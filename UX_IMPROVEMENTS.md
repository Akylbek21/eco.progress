# UX Improvements

## Что было улучшено

- Главная страница стала понятнее для нового клиента: простой заголовок, ясное описание услуг, быстрый путь к заявке и консультации.
- Добавлен блок выбора ситуации клиента: новый объект, проверка, отходы, отчетность, анализы или помощь специалиста.
- Популярные услуги переписаны человеческим языком без перегруза терминами.
- Добавлен блок "Как это работает" с пятью понятными этапами.
- Публичное меню упрощено до основных клиентских разделов.
- Footer стал полезнее: услуги, контакты, WhatsApp, email, юридическая информация и ненавязчивый вход для сотрудников.
- Добавлена плавающая WhatsApp-кнопка на публичных страницах.
- Форма заявки в кабинете клиента разделена на понятные шаги.
- Кабинет клиента получил приветственный блок, статистику, большую кнопку создания заявки и понятную статусную дорожку.
- FAQ заменен на реальные клиентские вопросы.
- Страница контактов получила карточки каналов связи и форму консультации.

## Как проверить результат

1. Запустить проект: `npm run dev`
2. Открыть главную страницу и проверить hero, блок ситуаций, популярные услуги и шаги.
3. Перейти в `/services` и проверить выбор конкретных работ внутри направлений.
4. Перейти в `/cabinet/orders/new` и проверить пошаговую форму заявки.
5. Проверить кабинет клиента `/cabinet`: статистику, последние заявки и статусную дорожку.
6. Проверить `/faq` и `/contacts`.
7. Проверить мобильную ширину 360-430px.
8. Выполнить `npm run build`.
